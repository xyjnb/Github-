---
初始化本地git库<br>
git init<br>
下载远程git库<br>
git clone https://github.com/fjyazzy/wxTest.git<br>
添加远程git库目标<br>
git remote add wx https://github.com/fjyazzy/wxTest.git<br>
把最新版本的代码库拖到本地<br>
git pull wx master<br>
---
修改本地代码<br>
---
Git add . (注意 "." 不能漏掉)<br>
git commit "修改说明"<br>
Git push wx master<br>
